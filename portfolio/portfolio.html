FRONTEND

Frontend development focuses on the part of web development that involves creating the visual and interactive aspects of a website or web application. It includes the use of HTML, CSS, and JavaScript.

WORDPRESS

WordPress is a popular content management system (CMS) used for creating websites and blogs.Wide range of plugins and themes are used to customize sites without extensive coding knowledge.
